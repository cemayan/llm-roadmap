# LLM-Era Roadmap — working repo

Everything produced while working through [ROADMAP.md](ROADMAP.md) lives here:
notes, experiments, and phase deliverables. The roadmap file in this repo is
the single source of truth for the plan.

## Layout

| Path | What goes here |
|---|---|
| `ROADMAP.md` | The plan itself — phases, checkboxes, traps, resources |
| `PROGRESS.md` | Weekly log: what got done, what's next, blockers |
| `phase-0-intuition/` | Karpathy/3B1B notes, nanoGPT experiment, "GPT by hand" post draft |
| `phase-1-primitives/` | Go agent loop, MCP server, RAG experiment, eval harness |
| `phase-2-product-llm/` | Pointers only — the real work lives in the entity-journey product repo |
| `phase-3-visibility/` | Post drafts, publish checklist |
| `notes/` | Cross-cutting notes that don't belong to a single phase |

## Rules (from the roadmap's Traps section)

- A phase is done when its **deliverable** exists — not when the videos are watched.
- No frameworks (LangChain etc.), no fine-tuning, no new resources beyond the short list.
- Phase 2 belongs to the OSS product plan (`workspace/todo.md`) — don't duplicate it here.
